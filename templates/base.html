About the project

This is a software project for SIH- Smart India Hackathon 2022.

The project is a web application that allows users to upload their handwritten documents and convert them into human-readable and editable text. We support most of the regional Indian languages including hindi, english, french, german and spanish.

This project is developed by six students of Vellore Institute of Technology, Chennai Campus.
